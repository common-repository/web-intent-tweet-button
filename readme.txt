Adds Wed Intent Tweet Button to posts.

== Description ==
Web Intent Tweet Button adds "Tweet Button" below every post of your WordPress powered site.
Web Intent provide popup optimized flow for your tweets. They make the users to interact with 
their followers without leaving the page, instead they provide popub based window for your tweets.
More information about Web Intents can be found here https://dev.twitter.com/docs/intents.

== Installation ==
1. Download the plugin
1. Unzip the plugin and upload to /wp-contents/plugins/
1. Navigate to "Plugins" panels.
1. Activate the plugin.
1. Navigate to Plugins Menu on your left sidebar and click Web Intent Tweet Button.
1. Enter your Twitter Name in the configuration page of the plugin.
